# Лабораторна робота №5 — Навігація з Expo Router

## Опис реалізованого функціоналу

- File-based маршрутизація з Expo Router
- Глобальний контекст авторизації (AuthContext)
- Публічні екрани: Вхід та Реєстрація
- Захищені маршрути з перенаправленням
- Каталог товарів з FlatList
- Динамічні маршрути для деталей товару
- Екран 404 для неіснуючих маршрутів

## Інструкція запуску

1. Встановити залежності:
npm install

2. Запустити проєкт:
npx expo start

3. Відсканувати QR-код у застосунку Expo Go на телефоні

## Висновки

1. Перенаправлення неавторизованого користувача реалізується через компонент Redirect у файлі _layout.jsx захищеної групи.
2. Link — декларативна навігація в JSX. router.push() — програмна навігація в функціях.
3. Динамічні маршрути створюються файлом [id].jsx. Параметри отримуються через useLocalSearchParams().
4. Глобальний контекст дозволяє будь-якому екрану мати доступ до стану авторизації без передачі через props.
5. Групи маршрутів логічно розділяють екрани. Назва групи в дужках ігнорується в URL.
