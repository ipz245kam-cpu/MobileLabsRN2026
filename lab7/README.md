# Лабораторна робота №7 — Redux Toolkit

## Інструкція запуску
1. Встановити залежності: `npm install --legacy-peer-deps`
2. Запустити: `npx expo start --lan`
3. Відсканувати QR-код в Expo Go

## Функціонал
- Каталог товарів з фото, описом та ціною
- Додавання товарів до кошика
- Зміна кількості та видалення товарів
- Оформлення замовлення з валідацією
- Історія замовлень
- Збереження стану через redux-persist

## Висновки
1. Глобальний стан — централізоване сховище даних для всіх компонентів
2. Redux Toolkit спрощує роботу з Redux через createSlice та configureStore
3. createSlice створює actions та reducer в одному місці
4. configureStore налаштовує Redux store з middleware
5. redux-persist зберігає стан після перезапуску додатку
