# Лабораторна робота №4 — Файловий менеджер

## Опис реалізованого функціоналу

- Навігація по директоріях з відображенням поточного шляху
- Створення папок та текстових файлів (.txt)
- Перегляд та редагування вмісту файлів
- Видалення файлів і папок з підтвердженням
- Перегляд детальної інформації про файл (назва, тип, розмір, дата)
- Навігація назад до попередньої директорії

## Інструкція запуску

1. Встановити залежності:
npm install

2. Встановити expo-file-system:
npx expo install expo-file-system

3. Запустити проєкт:
npx expo start

4. Відсканувати QR-код у застосунку Expo Go на телефоні
