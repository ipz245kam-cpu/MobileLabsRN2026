# Лабораторна робота №2 — React Native Navigation & Lists

## Що реалізовано
- FlatList з Pull-to-Refresh та підвантаженням при прокрутці
- SectionList з групами контактів
- Drawer Navigator з кастомним меню (аватар, ПІБ, група)
- Stack Navigator з передачею параметрів між екранами

## Структура
- `app/index.js` — навігація
- `app/screens/MainScreen.js` — екран новин
- `app/screens/DetailsScreen.js` — екран деталей
- `app/screens/ContactsScreen.js` — екран контактів
- `components/CustomDrawer.js` — кастомне меню

## Запуск
npm install
npx expo start
